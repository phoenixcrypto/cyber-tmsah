## 📦 التثبيت والتشغيل

### متطلبات النظام

- Node.js >= 18.0.0
- npm >= 8.0.0

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/phoenixcrypto/cyber-tmsah.git
cd cyber-tmsah

# تثبيت المكتبات
npm install

# تشغيل المشروع محلياً
npm run dev
```

افتح المتصفح على: `http://localhost:3000`

### النشر

```bash
# بناء المشروع
npm run build

# تشغيل الإنتاج
npm start
```

## 🔌 تكامل Strapi CMS

المشروع يدعم نظامين لمصدر البيانات:

### 1. **النظام الحالي** (افتراضي)
- قاعدة بيانات محلية في الذاكرة
- يعمل بدون أي إعداد إضافي
- مناسب للتطوير والاختبار

### 2. **Strapi CMS** (اختياري)
- نظام إدارة محتوى احترافي
- واجهة سهلة لإدارة المحتوى
- مناسب للإنتاج

#### إعداد Strapi

1. راجع دليل التكامل الكامل: [`STRAPI-MIGRATION.md`](./STRAPI-MIGRATION.md)
2. قم بإعداد Strapi وفقاً للدليل
3. أضف المتغيرات البيئية:

```env
NEXT_PUBLIC_STRAPI_URL=https://your-strapi-url.com
STRAPI_API_TOKEN=your_api_token_here
```

4. المشروع سيكتشف تلقائياً Strapi ويعتمد عليه

### نقل البيانات التلقائي

- **إذا كان Strapi متاح**: يستخدم Strapi
- **إذا كان Strapi غير متاح**: يعود للنظام الحالي تلقائياً
- **لا حاجة لإعادة تشغيل**: الانتقال سلس ومباشر
